# 📦 Projeto Backup de Arquivos em Python

Projeto desenvolvido em **Python** para realizar **backup automático de arquivos ou pastas**, facilitando a proteção de dados importantes.

## 🚀 Funcionalidades

* 📁 Backup de arquivos e diretórios
* 🗂 Organização automática dos backups
* ⚡ Execução rápida pelo terminal
* 🔒 Proteção contra perda de dados
